# React Cards using JSON & map()

This project demonstrates how to dynamically render cards in React using the `map()` function and data from a local JSON file.<br>

---

## 📁 Folder Structure

public/<br>
└── images/<br>
└── (card images)<br>
<br><br>
src/<br>
├── App.jsx # Main entry point<br>
├── Cards.jsx # Component that maps and displays cards<br>
├── Cards.css # Styles for the cards<br>
└── Cards.json # JSON file containing card data<br>


## 🚀 How to Run

1. Clone the repository or download the ZIP  <br>
2. Open terminal in the project folder  <br>
3. Install dependencies:<br>
--npm install<br>
4. Start the development server:<br>
--npm run dev<br>
5. Open http://localhost:5173/ in your browser<br>
<br><br>
